Le Swish Prophet

Data Analytics:

This statistics-based application is designed to predict the outcomes of NBA games. To optimize its accuracy, my application implements a datamining API to hack into the NBA's official database for up-to-date statistics. The data is then wrangled and processed with pandas dataframes. Finally, my predictive algorithm draws upon the mathematical principles of regression and weighted moving averages to draw based conclusions.

Bionic Visual Cortex

A.I. Vision:

This UX interface is designed for the image processing component of my segmented AI being, or neurologically the occipital lobe. Built on Convolutional Neural Networks for Deep Learning, my program loads a light-weight model from the Keras library. With the pre-processing of image files into multidimensional arrays, the application is able to carry out basic linear algebra to solve the complex problem of image classification using only 16MB of training data.
